Questions

Is it really free?

Yes. Praat is MIT-licensed open source. Download it, use it, share it with anyone. No subscription, no account, no paywall.

Does my voice get sent to a server?

No. Transcription happens locally on your device using Whisper models. Praat makes exactly one network call — to download the model file once on first launch. After that, it can run fully offline.

What do I do if macOS says the app is damaged?

macOS Gatekeeper blocks apps that aren't code-signed by a paid Apple Developer account. Right-click the Praat app → Open → Open Anyway. You only need to do this once. Or, from Terminal: xattr -cr /Applications/Praat.app

What about Windows SmartScreen?

When you run the installer, Windows may show "Microsoft Defender SmartScreen prevented an unrecognized app." Click "More info" → "Run anyway." Same reason — no paid code-signing cert attached.

What permissions does Praat need?

Microphone access (to hear you) and, on macOS, Accessibility (to paste transcribed text into other apps). Nothing else.

Which model should I pick?

Turbo is the best default for most machines — fast and accurate. Pick Small on older hardware and Large if you want the highest quality and have the GPU to spare.
